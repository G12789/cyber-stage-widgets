# Cyber Stage Widgets

> 一组**赛博朋克风格的桌面视觉特效小程序**，用 Python + tkinter 写成。
> 纯娱乐 / 短视频拍摄用途：界面看起来像很专业的「AI 大模型软件」，但**不联网、不做任何真实计算**，输出都是预设的、随机的或马赛克化的搞笑结果。

A small collection of **novelty cyberpunk-style desktop visual-effect apps** (Python + tkinter). They are for **fun and short-video content only** — no network, no real computation, outputs are fake/random by design.

## ⚠️ 说明 / Disclaimer
- 这些程序**没有任何真实功能**，不会处理、获取或存储任何真实数据。
- 所有"结果"都是写死的或随机生成的视觉效果（例如全是 `█` 或 `*` 占位符）。
- 仅用于制作有趣的短视频，请勿用于误导他人。
- These apps perform **no real function**. They never access or store real data. All "results" are hard-coded/random visual effects. For entertainment video content only.

## 包含的特效 / Widgets
| 程序 | 效果 |
|------|------|
| 翻译器 | 不管输入什么，永远"翻译"成同一个梗 |
| 亲戚称呼学习器 | 一本正经算出搞笑的亲戚称谓 |
| 生僻字发音器 | 假装 OCR+TTS，其实原样念回去 |
| 赛博解码特效 | 黑客风进度动画，结果永远是 `**********` |
| 赛博定位特效 | "机密"信息全是 `████` 马赛克，纯视觉演出 |

## 运行 / Run
需要 Windows + Python 3.10+。
1. 双击 `00_安装依赖.bat` 安装依赖（见 `requirements.txt`）
2. 双击任意 `启动_xxx.bat` 运行对应特效
3. 大部分窗口按 `ESC` 退出

## 技术 / Stack
Python 3 · tkinter · Pillow（部分弹窗素材）
